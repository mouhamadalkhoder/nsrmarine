【FEATURES】

NSR's new generation of AIS EPIRB & VDR FFC:

- Meet the latest IMO regulation – IMO MSC.471 (101).

- 64GB memory capacity to record at least 48 hours onboard data.

- 406MHz transmission together with 121MHz.homing & AIS transmitter.

- Highly efficient battery.

- Self-protective device used to avoid possible false alerts.

- Self-test function applicable.

- Easily coding/reading user data with infrared communication.

- RLS function available.

- GMDSS compliant.

- Global distress alerting.

- Fully enclosed housing.

- VDR floating capsule.

- COSPAS-SARSAT type approval.
